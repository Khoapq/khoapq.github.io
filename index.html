Create Bitbucket Pages / Free Hosting at BitBucket

An introduction

The slightly undocumented Bitbucket Pages feature allows for you to host simple HTML web pages from one of your Bitbucket repositories. Update your web page by simple pushing changes to the repository.

Creating your web page

Let’s say your username is khoapq. You need create a new repository, called khoapq.bitbucket.org.

As soon as you’ve pushed a document to your khoapq.bitbucket.org repository, it will show up at http://khoapq.bitbucket.io/

That is…

  1. Create a repository called yourusername.bitbucket.org
  2. Push pages to your repository
  3. See your site: http://yourusername.bitbucket.io/
